  • Attributes and Properties are not the same
  • Attributes are defined by html
  • Properties are defined by the DOM
  • Attributes initialize DOM properties and they they are done. Attribute values cannot change once they are initialized
  • Property values however can change
  • HTML attribute value specifies the initial value(value attribute)
  • DOM value property is current value(value property)
